## Recipe-finder — це невеликий React-додаток для пошуку рецептів.

Він використовує API Edamam для отримання рецептів за ключовим словом.

## Користувач може:

Ввести назву страви або інгредієнт у пошуковий рядок.
Натиснути Enter або кнопку пошуку.
Переглянути результати з картинками та списком інгредієнтів.
Додаток також має фонове відео та приємний інтерфейс.

## Технології

React (useState, useEffect)
React Icons (GoSearch)
Fetch API
CSS для стилізації
Edamam Recipe API
